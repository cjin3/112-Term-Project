Welcome to BTD! (Bad Tower Defense)

BTD is a tower defense game. 
Create your own maps with the map builder, and save them to save1 or save2.
Load your creations with the load menu.

Please run main.py.
Make sure that all images are in the same folder as the source code.

You can press the 'escape' key to go back to the previous scene.
You can press the 'r' key to go back to the title screen.

Note: parseMap(map) function is taken from https://github.com/vixu8/hack112, main.py, lines 473 to 484, hack112 project
